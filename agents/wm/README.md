# World-Model Notes

## English

- `INSTALL.md`: installation and submodule setup for Cosmos and related VLA/world-model repos.
- `COSMOS_INTEGRATION.md`: how `CosmosEnv` fits into `verl.experimental.vla`.
- `THIRD_PARTY_ROLE_ANALYSIS.md`: repo-by-repo analysis of `lingbot-vla`, `lingbot-va`, `dreamzero`, `Motus`, and Cosmos.
- `WORLD_MODEL_RL_FRAMEWORK.md`: design sketch for a dedicated world-model / embodied robotics RL framework and how it differs from `verl`.
- `VLA_RL_MODES.md`: Chinese quick-reference comparing VLA/VLM/text RL modes already present in the repo.
- `VLA_RL_MODES_ONEPAGE.md`: one-page cheat sheet for quick comparison and presentation.
